== Description ==

This plugin displays an allow on the right side of your blog. The position of the allow is fixed and whenever you click it, you move back to the top of the page.
Many thanks to Web-Graphics for the original Javascript.

== Installation ==

1. Upload `wp-modore` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the ÅePluginsÅf menu in WordPress

== Frequently Asked Questions ==

= Can I change the image of the arrow? =

Make a PNG format image, save as `modore.png` and put it in the `wp-modore` folder.
